## Installation

1. Clone the repository:
```bash
git clone https://github.com/hassandaher1/2026.git
cd 2026
```

2. Open `index.html` in your web browser

That's it! No build process or dependencies required.

## Usage

1. Click on any day in the calendar to add activities
2. Select activity types: Sport, Work, or Goals
3. Add comments or notes for each day
4. View your progress in the charts below

## Technologies

- Pure HTML, CSS, and JavaScript
- Chart.js for data visualization
- localStorage for data persistence
